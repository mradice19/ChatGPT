# Demo visual del rediseño

## Cómo visualizar localmente
```bash
python -m http.server 5500
```
Luego abrí: `http://localhost:5500`

## Cómo visualizar desde GitHub
1. Subí estos archivos al repo (`index.html` + `styles.css`).
2. En GitHub: **Settings → Pages**.
3. En **Source**, elegí **Deploy from a branch**.
4. Branch: `main` (o la rama publicada), carpeta `/ (root)`.
5. Guardá. En 1-3 minutos GitHub te muestra la URL pública.

Ejemplo de URL: `https://TU-USUARIO.github.io/NOMBRE-REPO/`
